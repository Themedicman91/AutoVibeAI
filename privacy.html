Privacy Policy

AutoVibeAI is committed to protecting your privacy. This Privacy Policy explains how we handle data while automating TikTok video posting.

1. Data Collection

We collect minimal data required to operate AutoVibeAI, including TikTok API access tokens and user authentication data. No personal user data (e.g., email, name) is stored.

2. Data Usage

Data accessed via TikTok API (e.g., trends, video metadata) is used solely for AI analysis and video generation. We use third-party services like OpenAI and Pika.art, which may process this data per their policies.

3. Data Sharing

We do not share your data with third parties except as required to operate the service (e.g., TikTok API, OpenAI, Pika.art). We comply with TikTok's API guidelines.

4. Data Security

We take reasonable measures to secure your data, but no system is 100% secure. You are responsible for maintaining the security of your TikTok account.

5. Changes to Policy

We may update this Privacy Policy at any time. Changes will be posted on this page, and your continued use constitutes acceptance of the updated policy.
